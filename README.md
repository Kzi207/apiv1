# apiv1

Simple TypeScript Express server exposing existing handlers under `/api/v1/`.

Quick start:

1. Install dependencies:

```bash
npm install
```

2. Run in development (requires `ts-node-dev`):

```bash
npm run dev
```

3. Example endpoints:

- GET /api/v1/media/anime
- GET /api/v1/media/girl
- GET /api/v1/capcut/down?url=...
