Hướng dẫn sử dụng API

Trang này hướng dẫn chi tiết cách sử dụng các API trong dự án này và cho phép bạn test trực tiếp.

Hướng dẫn sử dụng API

1. SoundCloud

  • Tìm kiếm nhạc: /api/v1/soundcloud/search?q=KEYWORD
  • Lấy link tải nhạc: /api/v1/soundcloud/download?url=LINK
  • Lấy link tải nhanh (nếu có): /api/v1/soundcloud/search?q=KEYWORD (trả về mp3 nếu tìm thấy)

2. TikTok

  • Tải video: /api/v1/tiktok/down?url=LINK

3. Facebook

  • Tải video: /api/v1/facebook?url=LINK
  • Lấy info video: /api/v1/facebook/getinfov2?url=LINK

4. Capcut

  • Tải template: /api/v1/capcut/down?url=LINK

5. Đố vui, truyện cười, thơ, ca dao, danh ngôn...

  • Đố vui: /api/v1/game/dovui
  • Truyện cười: /api/v1/media/truyencuoi
  • Thơ: /api/v1/media/du
  • Ca dao: /api/v1/media/cadao
  • Danh ngôn: /api/v1/media/chamngon
  • Hình ảnh anime/girl: /api/v1/media/anime, /api/v1/media/girl, /api/v1/media/girlsexy
  • Video anime/gái: /api/v1/media/videoanime, /api/v1/media/videogai
  • Thính, mông, joker: /api/v1/media/thinh, /api/v1/media/mong, /api/v1/media/joker

Cách sử dụng

  1. Chạy server Node.js: 
    npm start
  2. Truy cập các endpoint API đúng như ví dụ phía trên (bắt đầu bằng /api/v1/...).
  3. Có thể dùng trình duyệt, Postman, curl hoặc form bên dưới để test trực tiếp.
  4. Nếu dùng form bên dưới, chỉ cần nhập phần sau domain, ví dụ: /api/v1/soundcloud/search?q=son tung

Test API trực tiếp


        
        
Ví dụ sử dụng với curl

        
curl "http://localhost:3000/api/v1/soundcloud/download?url=https://soundcloud.com/abcxyz"
